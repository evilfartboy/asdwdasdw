# VIVET - Black Owned Clothing Brand

Welcome to VIVET, a modern and stylish clothing brand website celebrating Black excellence and community.

## 🎨 Features

- **Modern Design**: Sleek, premium black theme with gold and orange accents
- **Responsive Layout**: Works perfectly on desktop, tablet, and mobile devices
- **Product Showcase**: Featured clothing collection with product cards
- **Community Focus**: Highlights howyouknowog and other Black creators
- **Smooth Animations**: Professional fade-in and scroll effects
- **Contact Form**: Easy way for customers to get in touch
- **Social Media Integration**: Direct links to social platforms
- **Mobile-Friendly Navigation**: Hamburger menu for smaller screens

## 📁 Files

- `index.html` - Main website structure and content
- `styles.css` - Complete styling with animations and responsive design
- `script.js` - Interactive functionality and event handlers
- `README.md` - This file

## 🚀 How to Use

1. **Open the Website**: 
   - Right-click on `index.html` and select "Open with Default Browser"
   - Or double-click the file to open it in your default browser

2. **Navigation**: Use the navigation menu to jump between sections:
   - HOME - Hero section with CTA
   - SHOP - Product collection
   - ABOUT - Brand story and community support
   - CONTACT - Get in touch

3. **Interactive Features**:
   - Click "Shop Collection" button to jump to products
   - Add items to cart (demo functionality)
   - Fill out the contact form to send messages
   - Follow howyouknowog on Twitch link in the About section

## 🛠 Customization

To customize the website:
- **Colors**: Edit CSS variables in `styles.css` (--primary-color, --secondary-color, etc.)
- **Content**: Edit text in `index.html` sections
- **Products**: Add more product cards in the shop section
- **Images**: Replace icon placeholders with actual product images

## 📞 Getting Started

Simply open `index.html` in any modern web browser. No installation or server required!
